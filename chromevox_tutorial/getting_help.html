Getting help.

We've reviewed almost a dozen ChromeVox keyboard shortcuts now, but there's many, many more available to help you navigate efficiently. But you don't have to memorize them all! ChromeVox has a simple help menu where you can browse and search through the full list of ChromeVox commands.

To activate the ChromeVox command help menu, press ChromeVox period. You can then use the up or down arrow keys to browse through the available shortcuts, or start typing to narrow down the list. For example, if you're looking for the shortcut to take you to the next checkbox, typing "check" in the help menu will announce the command you're looking for. (It's ChromeVox N, X if you were curious). Pressing the down arrow will announce the command to take you to the previous shortcut (which is ChromeVox P, X).

Once you've found the command you want, pressing ChromeVox keys by themselves will close the menu.
