Navigation in Detail.

ChromeVox allows you to view a particular page at different levels of detail. What you have seen so far is group level. You can change the current level of detail using keyboard shortcuts. These shortcuts are ChromeVox minus key and ChromeVox equals key. ChromeVox equals will increase the level of detail - think of it as "zooming in" to content on the page. ChromeVox minus will decrease the level of detail, so you can get a broader overview of page content. You can think of this as "zooming out" of the page.

ChromeVox will announce the current detail level after you switch. The group level is the least detailed level. Character level is the most detailed level.

Let's talk about moving to an object. Object level is a more specific level of detail than group level. At the object level, you can move within groups to view individual web objects like links, buttons, headers, and edit fields. So once you navigate to a group that you want to explore in more detail, simply press ChromeVox with equals to zoom ChromeVox in to specific objects inside the group.

Try selecting the second listitem object within the next group by zooming into the group with ChromeVox equals. Then try zooming back out by pressing ChromeVox minus. Try this on the following group by first moving to it in group mode with ChromeVox down.

Did you see the second listitem object? Was it orange? Keep moving around until you feel comfortable.

Once you get a feel for how object and group navigation works, you can use this to quickly move around a page. If you are not interested in any links within a group, simply move past the group to the next. However, if you are interested in a link or other object within a group, press ChromeVox equals to zoom to the object detail level and explore the contents of the group.
