Installing ChromeVox on Mac

ChromeVox is an open source JavaScript screen reader designed for Chrome OS. Although ChromeVox is targeted for Chrome OS, it will also work under Chrome 14 (Beta) on Mac. ChromeVox is still BETA - there are definitely a few rough edges, but we are iterating rapidly to improve the product. These following instructions have been tested with VoiceOver and found to be working.

Once you have ChromeVox installed, press Command + F5 to switch off VoiceOver when you are navigating web content in Chrome with ChromeVox. When you want to go back to using VoiceOver, press Command + F5 again to turn VoiceOver back on. You can turn off ChromeVox by going to "Tools", "Extensions", and then disabling ChromeVox.

We have a mailing list for ChromeVox users here.

I. Installing Chrome

  1. Go to the Chrome Beta Download page.
  2. Click on the "Agree and download" button to start the download.
  3. Open the downloaded GoogleChrome.dmg file and copy Google Chrome to your Applications folder, just like you would when installing any other Mac program.

II. Installing ChromeVox Through the Chrome Web Store

The preferred method for installing ChromeVox is to install it through the Chrome Web Store.

  1. Start Chrome.
  2. Go to the ChromeVox page in the Chrome Web Store.
  3. Click on the "Add to Chrome" button.
  4. A dialog should pop up titled "Confirm Installation" that lists the permissions that ChromeVox needs. Please see our Permissions Guide if you would like more information about why ChromeVox needs these permissions. Click on the "Install" button to complete the installation process.
  5. Congratulations! You've installed ChromeVox. Now would be a good time to go through the ChromeVox tutorial.
