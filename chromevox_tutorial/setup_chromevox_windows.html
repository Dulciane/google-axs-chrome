Installing ChromeVox on Windows

ChromeVox is an open source JavaScript screen reader designed for Chrome OS. Although ChromeVox is targeted for Chrome OS, it will also work under Chrome 14 (Beta) on Windows. ChromeVox is still BETA - there are definitely a few rough edges, but we are iterating rapidly to improve the product. These following instructions have been tested with JAWS 12 and found to be working.

Once you have ChromeVox installed, you will be navigating web content in Chrome with ChromeVox and everything else with your existing screen reader. If you wish to go back to navigating web content with your existing screen reader, you can turn off ChromeVox by going to "Tools", "Extensions", and then disabling ChromeVox.

We have a mailing list for ChromeVox users here.

I. Installing Chrome

  1. Go to the Chrome Beta Download page.
  2. Find the "Accept and Install" button and press Enter to start the download.
  3. Run the "ChromeSetup.exe" file that you just downloaded.

II. Installing ChromeVox Through the Chrome Web Store

The preferred method for installing ChromeVox is to install it through the Chrome Web Store.

  1. Start Chrome.
  2. Go to the ChromeVox page in the Chrome Web Store.
  3. Find the "Add to Chrome" button and press Enter.
  4. A dialog should pop up titled "Confirm Installation" that lists the permissions that ChromeVox needs. Please see our Permissions Guide if you would like more information about why ChromeVox needs these permissions. Click on the "Install" button to complete the installation process.
  5. Congratulations! You've installed ChromeVox. Now would be a good time to go through the ChromeVox tutorial.
